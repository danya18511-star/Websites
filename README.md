Hello! As you can probably guessed, all the work was done by us with ChatGPT being used to help compile, arrange and comment on the programs so we acknowledge of it helping us to improve the program.  These are the instructions on how to run the program:
Download and unzip the package.

Open Google Chrome.

From Chrome, open File -> Open File... and select index.html (or drag-and-drop index.html into the browser).
Note: Chrome's file:// restrictions mean some fetches (remote JSON) may not run from file:// pages.

The app will prompt once for an encryption passphrase (on first operation that needs encryption).
That passphrase is stored in sessionStorage and will be required by all pages in the same browser session.
To change the passphrase, close the browser tab (or clear sessionStorage) and reopen the page to re-prompt.
